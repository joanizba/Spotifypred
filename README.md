### üß© 2. Creaci√≥n de Entorno Docker (Con Devbox)
Se adopta un nuevo archivo docker-compose.yml que incluye una imagen de Hadoop m√°s reciente y un contenedor devbox basado en ubuntu:latest para las herramientas.

*üìù Archivo docker-compose.yml*
```text

    image: ghcr.io/apache/hadoop:3.3.6 # Nueva imagen de Hadoop

      - my_network

    container_name: cassandra
    ports:
      - "9042:9042" # CQL port
    networks:
      - my_network

  nifi:
    image: apache/nifi:latest
    container_name: nifi
    ports:
      - "8080:8080" # NiFi UI
    environment:
      - NIFI_WEB_HTTP_PORT=8080
    networks:
      - my_network

  devbox: # Nuevo contenedor dedicado para herramientas
    image: ubuntu:latest # Una imagen base limpia y moderna
    container_name: devbox
    networks:
      - my_network
    stdin_open: true # Permite interactividad
    tty: true      # Asigna una TTY para una terminal interactiva

  my_network:
    driver: bridge
```
